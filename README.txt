# Blackbay Consulting — Static Site

## Quick preview (local)
1. Open `index.html` in any browser (double-click).
2. If you use VS Code: install the `Live Server` extension, then right‑click `index.html` → **Open with Live Server**.

## Publish on GitHub Pages
1. Create a new repo (e.g., `blackbay-consulting`).
2. Add/commit `index.html` to the repo root and push to `main`.
3. In repo **Settings → Pages**, set Source to **Deploy from a branch**, Branch **main / root**.
4. Wait ~1–2 min; your site will appear at the Pages URL.

## Custom domain
- In **Settings → Pages**, add your domain (e.g., `blackbay.fi`) and create the required DNS `A` / `CNAME` records at your registrar.
